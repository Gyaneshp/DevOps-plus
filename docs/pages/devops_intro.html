Why devops?

As we learned in the previous page, DevOps syntax can be executed by writing directly in the Command Line:

>> print("Hello, World!")
Hello, World!

Or by creating a devops file on the server, using the .py file extension, and running it in the Command Line:

C:\Users\Your Name>devops myfile.py

DevOps Indentation

Indentation refers to the spaces at the beginning of a code line.

Where in other programming languages the indentation in code is for readability only, the indentation in DevOps is very important. DevOps uses indentation to indicate a block of code.
